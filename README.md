Принимает переменные (env):
- PORT (Если 0 то TCP выключен)
- PATH

# Возможные сообщения

## Type 1 (Server)
Ошибка

## Type 2 (Server)
Лог с сервера

## Type 3 (Client)
Подписывается на логи сервера

## Type 4 (Client)
Шлёт команду на сервер

## Type 5 (Client)
Отключает сервер

## Type 6 (Client)
Запускает сервер

## Type 7 (Client)
Рестартит сервер

## Type 8 (Client)
Подписывается на получение ресурсов

## Type 9 (Server)
Мониторинги

## Type 10 (Server)
Первые мониторинги (сразу после Type 8), и потом не отправляются так как не меняются
